# apartment-furniture-planner (MVP v1)

Первая статическая версия без зависимостей (без React/Vite/TypeScript/npm). Проект — это HTML + CSS + JavaScript ES modules + SVG, готовый для запуска локально и через GitHub Pages.

## Запуск локально

```bash
python3 -m http.server 5173
```

Откройте: `http://localhost:5173`

## Запуск через GitHub Pages

1. Запушьте репозиторий в GitHub.
2. В `Settings -> Pages` выберите ветку (обычно `main`) и папку `/ (root)`.
3. Дождитесь публикации и откройте URL GitHub Pages.

## Где менять данные

- Геометрию квартиры (комнаты, стены, окна, проемы, инженерия): `data/apartment.js`
- Мебель и технику: `data/apartment.js` (`items`)
- Текстуры/цвета объектов: `assets/textures.js`

## Возможности MVP

- 2D SVG-план квартиры, масштаб в модели: метры, рендер: `1 м = 90 px`
- Стены разделены на несущие (`0.17 м`) и перегородки (`0.05 м`)
- Правая панель-дерево со связкой выбора «план ↔ дерево»
- Режим просмотра: только навигация
- Режим редактирования: перетаскивание мебели/техники
- Поворот выбранного movable-объекта на 15° клавишей `R`, на 90° кнопкой, и на произвольный угол через кнопку `Повернуть…`
- Zoom: колесо мыши и кнопки `+/-`
- Pan: перетаскивание пустого поля (pointer events, работает с мышью/тач/DeX)
- `Сброс вида`
- `Сброс мебели` к исходным координатам из `data/apartment.js`
- Кнопка `Размеры` для вкл/выкл размерных линий плана (между стенами, от угла до проемов, размеры проемов)
- Отдельная кнопка `Размеры мебели` для вкл/выкл габаритов мебели и техники
- Выравнивание выбранного movable-объекта: `Горизонтально` (0°) и `Вертикально` (90°)
- `К стене`: автопривязка выбранного объекта к ближайшей стене комнаты (только для объектов с `snapToWall: true`)
- Автосохранение положения мебели в `localStorage`
- `Экспорт мебели`: копирует в буфер блок `items: [...]` в формате `data/apartment.js` с текущими `x`, `y`, `rotation`

## Ограничения первой версии

- Комнаты описаны приближенными прямоугольниками
- Проемы и окна отображаются как упрощенные прямоугольники поверх стен
- Нет физики коллизий и реального вырезания геометрии стен
- Нет импорта плана из изображения и автопоиска контуров
- Нет backend, все данные и состояние локальные
